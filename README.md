# my-portfolio

url - https://my-portfolio-nine-cyan-23.vercel.app/

Personal portfolio site for Ye Min Aung — built with React, Vite, React Router, and GSAP.

## Features

- Home page with horizontal scroll showcase
- About page with contact form
- Projects page with creative work and GitHub repos
- Responsive layout

## Setup

```bash
npm install
npm run dev
```

Open `http://localhost:5173`.

## Build

```bash
npm run build
npm run preview
```
